# PrepEXAACSI

Proyecto estático para preparar el examen ACSI (21719) con:
- temario resumido
- formulario de fórmulas
- ejercicios resueltos
- test de autoevaluación

## Despliegue en Netlify

1. Sube este repositorio a GitHub.
2. En Netlify, crea un nuevo sitio desde GitHub.
3. Selecciona la carpeta raíz del proyecto.
4. Netlify detectará la web estática automáticamente.

El sitio se sirve desde el archivo `index.html`.
